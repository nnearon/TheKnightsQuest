# 🎮 The Knight's Quest

## 📌 Overview
**Game Title** is a 2D/3D game developed using the Godot Engine.  
The project was created as part of a coursework/assignment and focuses on gameplay mechanics, level design, and interactive systems.

---

## 🧩 Features
- Player movement and controls
- Enemy/obstacle mechanics
- Multiple levels 
- UI elements 
- Sound effects and background music
- Save/respawn system 

---

## 🕹️ How to Play
### Controls:
- Move: `WASD` / Arrow Keys
- Jump: `Space`

---

## ⚙️ Built With
- Godot Engine (version: [e.g. 4.2])
- GDScript
- Aseprite / Pixelart / Itch.io

---

## 🚀 How to Run the Project
1. Download or clone this repository
2. Open **Godot Engine**
3. Click **Import**
4. Select the `project.godot` file
5. Run the "StartingScreen"

---

## 🧠 Learning Outcomes
Through this project, the following concepts were applied:
- Scene system in Godot
- Signal handling
- Collision detection
- Basic game loop logic
- UI design and interaction
